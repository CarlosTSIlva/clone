# Template

## Instala as dependencias do projeto e os pacotes. 
```
yarn install
```

## Instala algum pacote expecifico. 
```
yarn add "pacote"
```

### Rodar o projeto. 
```
yarn serve
```

### Estados Vue 
Todo estado fica salvo dentro do Vue, e assim é possível manipular a dom utilizando o javascript.
